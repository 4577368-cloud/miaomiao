# 喵汪管家 (Meow Woof Butler)

## 1. 项目概述
- **核心价值**：解决养宠人士在出差、旅游或忙碌时无法照顾宠物的痛点。连接“需要服务的主人”与“提供服务的宠托师”。
- **对标产品**：溜达达、宠托师、美团/大众点评宠物上门服务板块。
- **差异化**：更优的定价透明度、更精准的地理位置匹配、更流畅的订单流程。

## 2. 用户角色
- **雇主 (主人)**：发布需求、支付、查看轨迹、评价。
- **接单员 (宠托师)**：接单、打卡、反馈、提现 (需实名/背景审核)。

## 3. 核心功能 (MVP)
### 3.1 雇主端
- **智能发布向导**：分步引导 (服务类型 -> 宠物类型 -> 时长/频次 -> 附加服务)。
- **动态定价**：
  - 宠物类型系数 (中型犬+20%, 大型+40%等)
  - 服务时长加价
- **服务执行**：地理位置、宠物档案、在线支付。

### 3.2 接单员端
- **接单/抢单**：LBS匹配、抢单模式。
- **服务执行**：上门打卡、服务拍照/录像、服务项确认、离场打卡。
- **收入管理**：明细、提现。

## 4. 设计规范
### 品牌色
- **主色 (活力橙)**: `#FF8E3C`
- **辅助色 (肉垫粉)**: `#FFD1DC`
- **背景色 (奶油米白)**: `#FFFBF5`
- **文字主色 (暖深灰)**: `#4A423C`

### 交互
- **卡片式布局**: 白色背景 + 极浅粉色投影 `rgba(255, 209, 220, 0.4)`
- **圆角**: 16px - 24px
- **按钮**: 全圆角 (Capsule)

## 5. 进度规划
- **Phase 1 (MVP)**: 核心功能 (发布、接单、支付、评价) + 微信小程序。
- **Phase 2**: 优化 (动态定价、智能匹配、后台)。
- **Phase 3**: 扩展 (保险、社区、企业服务)。
